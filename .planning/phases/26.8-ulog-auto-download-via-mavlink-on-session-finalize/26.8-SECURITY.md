# Phase 26.8 — Security

> Per-phase security contract: STRIDE register across 8 plans, accepted risks, and audit trail.

---

## Trust Boundaries

| Boundary | Description | Data Crossing |
|----------|-------------|---------------|
| FC (MAVLink autopilot) → worker | Untrusted FC can send oversized size fields, unsolicited LOG_DATA, gap-spam, spoofed autopilot bytes | `LOG_ENTRY`, `LOG_DATA`, `HEARTBEAT` |
| Worker → `roz.v1.ArtifactService` | Inherits Phase 26.7 mitigations: tenant scoping, digest verification, path-traversal guard | `UploadArtifact` stream |
| `UlogConfig` → `finalize_ulog_archive` | Operator-trusted; bounded by outer 60s `tokio::time::timeout` wrap | `enabled`, `download_timeout_secs`, `keep_fc_copy` |
| `session_relay` → `finalize_ulog_archive` | Session-scope spawn; `Option<Arc<MavlinkBackend>>` gate at spawn site | backend clone, session_id, artifact_client |
| Worker → FC `LOG_ERASE` | One-way fire-and-forget; strict gate on verified upload | MAVLink msg 121 |
| Mock transport ↔ test fixture | Test-only trust boundary; fixture digest pinned | `.ulg` bytes |
