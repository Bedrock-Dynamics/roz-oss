# 26.7-09 — SC6 ArtifactService integration test

## Outcome
`crates/roz-server/tests/artifact_copper_roundtrip.rs` landed with 3 `#[ignore]`-gated, `test-helpers`-feature-gated integration tests. All 3 pass against a live Postgres testcontainer in 4.45s.

## Test list

1. **`upload_and_download_roundtrip`** — happy path. 128 KB deterministic fixture. Validates:
   - Server-observed `size_bytes` equals declared size.
   - `roz_session_artifacts` row exists with correct artifact_type, digest, size, tenant_id, session_id.
   - Exactly one non-`.tmp` file under `artifact_dir/{tenant_id}/{session_id}/` (rename succeeded).
   - Download stream re-assembles to the original bytes; final-frame `digest_sha256` matches.

2. **`tampered_digest_rejected_and_cleaned_up`** — D-34 negative path. 64 KB fixture with forged all-zero digest. Validates:
   - Server returns `Code::InvalidArgument`.
   - `walk(artifact_dir)` returns zero files (both `.tmp` and canonical cleaned up).

3. **`list_session_artifacts_empty_for_new_session`** — ListSessionArtifacts smoke on a fresh session UUID; asserts empty `artifacts[]`.

## Container startup time
Observed ~4.45s total for the 3-test sequential run, including one container boot (the `pg_container()` guard is `std::mem::forget`-ed, so the container is reused across all 3 tests in a single `#[ignore]`-`--ignored --test-threads=1` invocation).

## Harness differences vs `observability_export_grpc.rs`
- One helper only: `setup_harness()` (no `setup_harness_for(tenant_id)` variant). Each test gets a fresh harness + fresh tenant row.
- Harness does not own `mcap_tmp` — it owns `artifact_tmp`.
- `ArtifactServiceImpl::new(pool, artifact_dir)` replaces `ObservabilityServiceImpl::new(pool, mcap_dir)`.
- Otherwise identical: same `InjectState` + `inject_extensions_middleware`, same `pg_container` + `std::mem::forget(guard)`, same `tempfile::tempdir + canonicalize`, same `tonic::service::Routes::new(server).prepare().into_axum_router().layer(...)`, same `TcpListener::bind("127.0.0.1:0")` + retry-loop `Endpoint::from_shared`.
