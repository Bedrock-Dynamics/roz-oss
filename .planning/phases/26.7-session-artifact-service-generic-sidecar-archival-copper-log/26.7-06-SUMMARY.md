# 26.7-06 — Worker copper_archive hook + ArtifactServiceClient wiring

## Outcome
Session-end copper archival scaffolding landed. Every edge session spawns `finalize_copper_archive` as a bounded (60s) background task immediately after the mcap_relay drain. The uploader is soft-fail per D-16 — never panics, never returns `Err`; every failure path is a `tracing::warn!` with structured fields.

2. **Scope?** Per-task (task dispatch), NOT per-edge-session. `handle_edge_session` does not construct a CopperHandle today.
3. **Dropped before finalize insertion?** N/A — no per-edge-session CopperHandle exists. `finalize_copper_archive` therefore typically finds an empty `{data_dir}/sessions/{session_id}/` and returns `Ok(())` with a debug log.
